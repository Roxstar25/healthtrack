# HealthTrack — Personal Health Metrics Tracker

A full-stack health tracking application built from scratch. Log your weight, sleep, workouts, and visualize your progress over time.

## Features

- User authentication (JWT + bcrypt)
- Daily health metrics logging
- Interactive charts with Chart.js
- Data persistence with SQLite + Prisma ORM
- RESTful API with Express.js
- Vanilla JavaScript SPA frontend

## Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript, Chart.js
- **Backend:** Node.js, Express.js
- **Database:** SQLite, Prisma ORM
- **Authentication:** JWT, bcryptjs
